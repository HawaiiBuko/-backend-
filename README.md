# 832301208_contact_backend

## 项目概述
本项目是联系人管理系统的后端服务，提供 RESTful API 支持联系人的增删改查操作，采用 Flask 框架和 SQLite 数据库，确保数据持久化和接口稳定性。

## 技术栈
- Python 3.8+
- Flask（Web 框架）
- Flask-CORS（跨域支持）
- SQLite（轻量级数据库）

## 功能说明
- **获取所有联系人**：支持分页和过滤（本版本暂为全量查询）
- **添加联系人**：创建新的联系人记录
- **编辑联系人**：更新已有联系人的信息
- **删除联系人**：移除指定联系人记录
